## Sobre

Esse é um projeto de estudos, com uma proposta diferente, que é construir um software que simule entraves encontrados no desenvolvimento de softwares utilizados por empresas no mundo real.

Ao invés de criar diversos projetos pequenos e isolados, decidi concentrar meus estudos e técnicas em um único sistema, evoluindo continuamente a sua arquitetura, regras de negócio e qualidade técnica de acordo com a minha evolução.

---

## Objetivos

Este projeto tem como objetivo praticar e consolidar conhecimentos em:

- Arquitetura de Software
- Desenvolvimento Full Stack
- Java e Spring Boot
- React e TypeScript
- Modelagem de Banco de Dados
- APIs REST
- Autenticação e Autorização
- Testes automatizados
- Docker
- CI/CD
- Documentação técnica
- Boas práticas de Engenharia de Software
